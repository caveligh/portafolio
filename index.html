BI Publisher

Es una herramienta 100% web que permite la creación, gestión y distribución de informes de una forma rápida y sencilla.


  • Funciona 100% web.
  • Acceso a los datos por parte de un usuario sin necesidad de instalar diversas aplicaciones con acceso a diversos orígenes de datos.
  • Seguridad de acceso a la información.
  • Acceso de datos por parte de los usuarios de forma no-invasiva (generación de informes sin modificar las aplicaciones existentes).
  • Formato origen de datos de cualquier tipo: SQL, Web Service, Excel, .XML, etc…
  • Formato de salida de datos: Interactivo, PDF, Excel, Word, HTML, etc…
  • Programación de informes con selección del destino.
  • Distribución de informes en función de las necesidades del destinatario (bursting).
  • Acceso a informes utilizando dispositivos móviles.
